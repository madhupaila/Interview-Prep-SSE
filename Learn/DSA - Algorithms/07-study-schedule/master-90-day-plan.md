# 90-Day DSA Mastery Plan

**Goal:** Pattern recognition at your fingertips + ~310 solved problems across all 26 patterns, ready for Senior SWE coding rounds.

**Daily commitment:** 2–3 hours (1 hr learning, 1–2 hrs solving). Each day: [15-min drill](daily-drill-protocol.md) first.

> If your interview is in <4 weeks, switch to the [Tier-A 4-Week Sprint](tier-A-4-week-sprint.md).

## Phase 2 — Tier B Depth (Days 46–70)

Cycle through patterns again at medium/hard depth. ~5 questions/day from [Tier B](../04-tier-B-comprehensive/).

| Days | Theme |
|------|-------|
| 46–50 | Arrays, windows, intervals (B) |
| 51–55 | Binary search, heaps, linked lists (B) |
| 56–62 | Trees, graphs, topo, union-find (B) |
| 63–67 | Backtracking, DP (B) |
| 68–70 | Greedy, bit, trie, strings, math (B) |

**Weekly:** one 45-min timed mock (random Tier A/B question).
**Deliverable:** Comfortable with medium problems; recognizing tie-breakers (DP vs greedy, BFS vs Dijkstra).

---

## Phase 3 — Tier C Mastery + Mocks (Days 71–90)

~4 hard questions/day from [Tier C](../05-tier-C-mastery/) + mixed mocks.

| Days | Focus |
|------|-------|
| 71–78 | Hard DP, hard graphs, design (LRU, median stream, trie board) |
| 79–84 | Mixed pattern recognition drills (random problems) |
| 85–88 | Company-specific sets — [Company Index](../06-company-question-index.md) |
| 89 | Full 2-question timed mock (45 min each) |
| 90 | Light review — flashcards + poster only |

**Deliverable:** Can solve unseen medium problems end-to-end with optimal complexity, and approach hards methodically.
