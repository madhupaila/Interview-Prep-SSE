# Tier-A 4-Week Sprint

For when an interview is **2–4 weeks away**. Covers the essential patterns + ~117 Tier A questions. Skip nothing in Week 1.

**Daily:** [15-min drill](daily-drill-protocol.md) + 2 hours solving.

## Non-negotiables

- Speak the pattern name before coding every problem.
- Always state complexity.
- Do the daily flashcard drill — it builds the recognition reflex.
