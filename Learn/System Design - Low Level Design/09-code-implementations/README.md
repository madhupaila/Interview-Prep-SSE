# Java Code Implementations

**Package convention:** `com.lldprep.{track}.{problem}`

## Structure

```
java/
├── classic/     # Classic OOD problems
├── patterns/    # Design pattern problems
├── concurrency/ # Concurrency LLD
└── genai/       # Gen AI LLD
```

## Tiers

| Tier | Count | Description |
|------|-------|-------------|
| Full | 25 | Working service + domain classes |
| Skeleton | 85 | Package stubs with TODO |

## Full Implementations

Parking Lot, Elevator, LRU Cache, Rate Limiter, Vending Machine, Chess, Deck of Cards,
Logger, Task Scheduler, Splitwise, Movie Booking, Uber Ride, ATM, Coffee Machine,
File System, Meeting Scheduler, Snake & Ladder, Tic-Tac-Toe, Parking Garage,
Strategy Payment, Command Undo, Observer Stock, State Vending, Producer-Consumer, RAG Orchestrator

## Run

Each package is self-contained. Compile with:

```bash
javac com/lldprep/classic/parkinglot/*.java
```
